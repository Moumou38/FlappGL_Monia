# FlappGL_Monia
IMAC3 OpenGL project demoscene

# Instructions : <br>

1- download the archive <br>
2- go to the project directory <br>
3- cd build <br>
4- cmake .. <br>
5- make <br>
6- cd bin <br>
7- ./FlappGL <br><br>

video : https://www.youtube.com/watch?v=aACwUTN7l1Y <br>

Press A to display the deferred rendering
